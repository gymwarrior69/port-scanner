# Simple Port Scanner

Ένα εκπαιδευτικό port scanner γραμμένο σε Python, φτιαγμένο για να κατανοήσω
πώς δουλεύουν τα TCP sockets και πώς λειτουργεί ένα βασικό network scanning tool.

## Λειτουργίες

- Σκανάρισμα ενός host σε εύρος ports (TCP connect scan)
- Σκανάρισμα ολόκληρου δικτύου (πολλά hosts, με κοινό prefix π.χ. `192.168.1.x`)
- Αναγνώριση κοινών services (HTTP, SSH, FTP, κλπ.) μέσω lookup table

## Πώς λειτουργεί

Χρησιμοποιεί τη built-in βιβλιοθήκη `socket` της Python και κάνει TCP
`connect()` σε κάθε port — αν η σύνδεση πετύχει, το port θεωρείται ανοιχτό.
Αυτό αντιστοιχεί σε ό,τι το Nmap αποκαλεί "TCP connect scan" (`-sT`).

## Χρήση

```bash
python3 port_scanner.py
```

Το πρόγραμμα θα σου ζητήσει να επιλέξεις:
1. Σκανάρισμα ενός host, ή
2. Σκανάρισμα ολόκληρου δικτύου

και μετά να δώσεις host/IP εύρος και port εύρος.

## Σημαντική σημείωση

Χρησιμοποίησέ το **μόνο** σε δικά σου συστήματα/δίκτυα, ή σε συστήματα όπου
έχεις ρητή άδεια να κάνεις δοκιμές (π.χ. δικό σου τοπικό δίκτυο, VM lab,
ή νόμιμα CTF/training περιβάλλοντα όπως TryHackMe / HackTheBox).
Το σκανάρισμα ports σε δίκτυα τρίτων χωρίς άδεια μπορεί να είναι παράνομο.
